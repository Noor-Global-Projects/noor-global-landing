# Noor Landing Page

A beautiful, modern landing page built with Vue 3 and Tailwind CSS.

🌐 **Live Site:** [https://noor.global](https://noor.global)

## Features

- 🚀 Vue 3 with Composition API
- 🎨 Tailwind CSS for styling
- 📱 Fully responsive design
- ⚡ Fast development with Vite

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
